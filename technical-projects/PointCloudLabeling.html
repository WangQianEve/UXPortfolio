Point Cloud Labeling
Patent No. CN201810075279.8 @ Pony.AI

A machine learning-empowered interaction method that helps people rapidly labeling point cloud data for autonomous driving technologies. We utilized machine learning to support multiple frame and multiple object labeling. This is an Qt App with OpenGL.
